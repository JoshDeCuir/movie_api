Movie and User API Documentation

Movies

Request URL Response
GET /movies Get a list of all movies
GET /movies/(specific title you are looking for by movie title) Get details of a specific movie
GET /movies/genre/(specific genre you are looking for by genre) Get details of a genre
GET /movies/director/(specific director you are looking for by name) Get details of the director of the movie

Users

Request URL Response
POST /users Create a new user
PUT /users/(specific id you are looking for in the array) Able to update users name
POST /users/(specific id you are looking for in the array)/(specific movie you want to add to the users favorite list) Adding a movie to a users favorites list
DELETE /users/:id/(specific movie you want to delete) Delete a specific movie from the users favorites list
DELETE /users/:id Delete specific user of the array
